# Face Icon Maker MVP開発タスク

## 1. 目的

MVPのゴールは以下とする。

```text
集合写真をアップロード
↓
顔を自動検出
↓
対象者を選択
↓
アイコン生成
↓
PNG保存
```

最小機能で価値を提供することを優先する。
